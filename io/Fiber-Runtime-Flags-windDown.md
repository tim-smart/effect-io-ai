# windDown

Returns `true` if the `WindDown` `RuntimeFlag` is enabled, `false`
otherwise.

To import and use `windDown` from the "Flags" module:

```ts
import * as Flags from '@effect/io/Fiber/Runtime/Flags'

// Can be accessed like this
Flags.windDown
```

**Signature**

```ts
export declare const windDown: (self: RuntimeFlags) => boolean
```
