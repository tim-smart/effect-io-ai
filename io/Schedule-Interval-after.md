# after

Construct an `Interval` that includes all time equal to and after the
specified start time.

To import and use `after` from the "Interval" module:

```ts
import * as Interval from '@effect/io/Schedule/Interval'

// Can be accessed like this
Interval.after
```

**Signature**

```ts
export declare const after: (startMilliseconds: number) => Interval
```
