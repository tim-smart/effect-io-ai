# isEmpty

Returns `true` if the specified `RuntimeFlagsPatch` is empty.

To import and use `isEmpty` from the "Patch" module:

```ts
import * as Patch from '@effect/io/Fiber/Runtime/Flags/Patch'

// Can be accessed like this
Patch.isEmpty
```

**Signature**

```ts
export declare const isEmpty: (patch: RuntimeFlagsPatch) => boolean
```
