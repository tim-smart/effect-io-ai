# before

Construct an `Interval` that includes all time equal to and before the
specified end time.

To import and use `before` from the "Interval" module:

```ts
import * as Interval from '@effect/io/Schedule/Interval'

// Can be accessed like this
Interval.before
```

**Signature**

```ts
export declare const before: (endMilliseconds: number) => Interval
```
