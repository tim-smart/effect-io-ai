# isNonEmpty

Returns `true` if the specified `Interval` is non-empty, `false` otherwise.

To import and use `isNonEmpty` from the "Interval" module:

```ts
import * as Interval from '@effect/io/Schedule/Interval'

// Can be accessed like this
Interval.isNonEmpty
```

**Signature**

```ts
export declare const isNonEmpty: (self: Interval) => boolean
```
