# OpenAiTelemetryAttributes

The attributes used to describe telemetry in the context of Generative
Artificial Intelligence (GenAI) Models requests and responses.

{@see https://opentelemetry.io/docs/specs/semconv/attributes-registry/gen-ai/}

To import and use `OpenAiTelemetryAttributes` from the "OpenAiTelemetry" module:

```ts
import * as OpenAiTelemetry from "@effect/ai-openai/OpenAiTelemetry"
// Can be accessed like this
OpenAiTelemetry.OpenAiTelemetryAttributes
```
