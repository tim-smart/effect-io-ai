## remove

Remove the entry for the specified key in the `HashMap` using the internal
hashing function.

```ts
declare const remove: { <K>(key: K): <V>(self: HashMap<K, V>) => HashMap<K, V>; <K, V>(self: HashMap<K, V>, key: K): HashMap<K, V>; }
```
