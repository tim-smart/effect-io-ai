## JsonSchema.Type

The JSON Schema primitive type names.

**When to use**

Use to restrict a JSON Schema `type` keyword to the supported primitive names.

**Signature**

```ts
type Type = "string" | "number" | "boolean" | "array" | "object" | "null" | "integer"
```
