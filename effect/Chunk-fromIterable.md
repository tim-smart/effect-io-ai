# fromIterable

Converts from an `Iterable<A>`

To import and use `fromIterable` from the "Chunk" module:

```ts
import * as Chunk from 'effect/Chunk'

// Can be accessed like this
Chunk.fromIterable
```

**Signature**

```ts
export declare const fromIterable: <A>(self: Iterable<A>) => Chunk<A>
```
