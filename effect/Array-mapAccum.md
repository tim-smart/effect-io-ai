## Array.mapAccum

Statefully maps over the chunk, producing new elements of type `B`.

**Example**

```ts
import { Array } from "effect"

const result = Array.mapAccum([1, 2, 3], 0, (acc, n) => [acc + n, acc + n])
console.log(result) // [6, [1, 3, 6]]
```
