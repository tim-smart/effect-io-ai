## Simplify

Simplifies the type signature of a type.

**Example**

```ts
import type { Types } from "effect"

type Res = Types.Simplify<{ a: number } & { b: number }> // { a: number; b: number; }
```

**Signature**

```ts
type Simplify<A> = {
  [K in keyof A]: A[K]
} extends infer B ? B : never
```
