# Monoid

`string` monoid under concatenation.

The `empty` value is `''`.

To import and use `Monoid` from the "String" module:

```ts
import * as String from '@effect/data/String'

// Can be accessed like this
String.Monoid
```

**Signature**

```ts
export declare const Monoid: monoid.Monoid<string>
```
