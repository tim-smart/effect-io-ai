## HttpMethod.all

Provides a readonly set containing every supported `HttpMethod` literal.

**When to use**

Use when you need to iterate over or test membership against every supported
HTTP method literal.

**Signature**

```ts
declare const all: ReadonlySet<HttpMethod>
```
