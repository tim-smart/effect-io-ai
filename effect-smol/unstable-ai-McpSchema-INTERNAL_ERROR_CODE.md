## McpSchema.INTERNAL_ERROR_CODE

Represents the JSON-RPC error code for internal server errors.

**When to use**

Use when building an MCP/JSON-RPC error response for an unexpected
server-side failure.

**Signature**

```ts
declare const INTERNAL_ERROR_CODE: -32603
```
