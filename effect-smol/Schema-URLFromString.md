## Schema.URLFromString

A transformation schema that decodes a `string` into a `URL`.

Decoding:
- A **valid** URL `string` is decoded as a `URL`

Encoding:
- A `URL` is encoded as a `string`

**Signature**

```ts
declare const URLFromString: URLFromString
```
