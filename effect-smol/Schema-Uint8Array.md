## Schema.Uint8Array

A schema for JavaScript `Uint8Array` objects.

**Default JSON serializer**

The default JSON serializer encodes Uint8Array as a Base64 encoded string.

**Signature**

```ts
declare const Uint8Array: Uint8Array
```
