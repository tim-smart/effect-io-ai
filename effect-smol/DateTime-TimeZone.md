## DateTime.TimeZone

Represents a time zone used by `DateTime.Zoned`.

A `TimeZone` is either a fixed offset from UTC or a named IANA time zone.

**Signature**

```ts
type TimeZone = TimeZone.Offset | TimeZone.Named
```
